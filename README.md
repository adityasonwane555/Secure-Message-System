# Secure Message System (C++)

A simple C++ program that demonstrates basic encryption and decryption using a combination of Caesar Cipher and XOR techniques.

Features

- Encrypt messages before sending
- Decrypt messages on receiving
- Simple key-based symmetric encryption
- Menu-driven interface

Concepts Covered

- Basic cryptography
- Data transformation
- Symmetric encryption

How to Run

1. Compile the code:
   g++ file.cpp -o app
2. Run:
   ./app

Note

This is a learning project and not secure for real-world use.
